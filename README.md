# 🎵 Smart Music App

A mood-based music streaming web app built using Flask.

## 🚀 Features

* 🎧 Mood-based music recommendation
* 🔁 Multiple songs per mood
* ⏭ Next button for playlist navigation
* 🌐 YouTube embedded player

## 🛠 Tech Stack

* Python (Flask)
* HTML, CSS
* YouTube Embed API

## ▶️ How to Run

```bash
python app.py
```

Then open:
http://127.0.0.1:5000/
